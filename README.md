## 📝 Descrição
O **Bridger Autofisch** é um macro de pesca automatizado e inteligente, desenvolvido em **AutoHotkey v2.0**. Ele utiliza reconhecimento de imagem e cor para identificar peixes, baús e resolver minigames de teclas de forma autônoma.

[cite_start]O projeto conta com uma **Interface (HUD)** elegante e flutuante para que você possa acompanhar o estado do macro em tempo real.

---

## ✨ Funcionalidades
- [cite_start]✅ **Auto-Fishing:** Detecta a fisgada e recolhe o peixe automaticamente[cite: 12, 13].
- [cite_start]⌨️ **Minigame Solver:** Identifica e pressiona as teclas (F, G, R, T) durante o minigame[cite: 14, 15].
- [cite_start]📦 **Detector de Baús:** Identifica a captura de baús e realiza a sequência de comandos para abrir/recolher[cite: 7, 8, 9].
- [cite_start]💎 **Rare Item Alert:** Notificação via ToolTip ao encontrar itens raros[cite: 10].
- [cite_start]🖥️ **HUD Customizado:** Interface minimalista que exibe o estado atual (ATIVO/DESLIGADO)[cite: 2, 4].

---

## ⚙️ Pré-requisitos e Recomendações
Para o funcionamento perfeito do macro, siga estas diretrizes:

### [cite_start]Configuração In-game[cite: 1]:
* **Vara de Pesca:** Deve estar no slot **'9'**.
* **Isca (Bait):** Deve estar no slot **'0'**.
* **Gráficos:** Mantenha o gráfico em **3 ou menos**.
* **Posicionamento:** Olhe diretamente para a água e evite lugares muito altos ou com relevo irregular.

### Dependências Técnicas:
* [AutoHotkey v2.0+](https://www.autohotkey.com/) instalado.
* [cite_start]Biblioteca `FindText.ahk` no mesmo diretório do script[cite: 1].

---

## 🚀 Como Usar
1.  Baixe o arquivo `BridgerFisch-v2.2.ahk`.
2.  Certifique-se de que o `FindText.ahk` está na mesma pasta.
3.  Execute o script.
4.  No jogo, utilize os atalhos:
    * [cite_start]**`F1`**: Liga/Desliga o Macro (Toggle).
    * [cite_start]**`F2`**: Reinicia o script ou sai com segurança.
