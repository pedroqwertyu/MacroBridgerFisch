AHK v2.0 FindText PixelSearch USE POR SUA CONTA E RISCO

⚠ ANTES DE USAR — LEIA ISSO

  • Coloque sua vara de pescar no slot 9
  • Coloque sua isca (bait) no slot 0
  • Olhe diretamente para a água antes de ativar
  • Evite pescar com o gráfico acima de 3
  • Evite pescar em lugares altos ou com muito relevo
  • Evite locais sem espaço suficiente ao redor

Dependências

AutoHotkey v2.0

Runtime necessário para executar o script

FindText.ahk

Biblioteca de reconhecimento visual de texto/imagem na tela

Como usar

01
Instale o AutoHotkey v2.0

Baixe em autohotkey.com e instale normalmente.

02
Coloque o FindText.ahk na mesma pasta

O arquivo FindText.ahk deve estar junto com o script principal.

03
Configure os slots no jogo

Slot 9 → Vara de pescar  ·  Slot 0 → Isca (bait)

04
Execute o script com o jogo aberto

Clique duas vezes no .ahk ou rode via AutoHotkey. O HUD aparecerá no centro da tela.

05
Posicione o personagem e ative

Olhe para a água, pressione F1 e o macro começa a pescar automaticamente.

Controles

F1
Ativar / Desativar o macro
O HUD mostra o estado atual em verde (ATIVO) ou vermelho (DESLIGADO)
F2
Recarregar o script
Reinicia o macro do zero, útil se travar

O que o macro faz

Prepara a vara

Abre o inventário, equipa a vara e posiciona a câmera olhando para a água.

Lança a isca

Equipa a bait, clica para lançar e aguarda a bolinha vermelha aparecer na água.

Detecta o peixe

Monitora a região da bolinha via PixelSearch. Quando o peixe morde, clica automaticamente para puxar.

Minigame de teclas

Usa FindText para detectar e pressionar as teclas F, G, R, T que aparecem na tela durante o minigame.

Detecta baú raro

Se um baú aparecer após o minigame, o macro se move até ele, interage e abre automaticamente.

Repete o ciclo

Volta ao início e pesca novamente enquanto o macro estiver ATIVO.
